# Travel Africa RAG Template

A starter front-end template for a Travel Africa RAG class project. The repository currently contains a static HTML5/CSS3 landing page that students can connect to their own retrieval-augmented generation back end.

## Folder structure

```text
TravelAfricaRAGProject/
├── README.md
├── static/
│   └── css/
│       └── styles.css
└── templates/
    └── index.html
```

## Run locally

From the repository root, start a simple static server:

```bash
python3 -m http.server 8000
```

Then open <http://localhost:8000/templates/> in your browser.
